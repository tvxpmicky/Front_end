인터뷰 스크립트
*기본프로필
- A팀 시각디자인학과 13학번 민소영

*주제1:여행의 목적
- 여행지 선정할때 기준은 사진은 볼거리가 많고 사진이 잘나오는 배경이 좋은곳이며 특히 맛집이 중요.
- 여행의 테마가 유적지라면 무서운느낌이 들어 처음엔 꺼려지나 막상 가보면 멋있는듯함.
- 선호하는 여행테마는 꽃놀이이며, 벚꽃을 보며 화사함을 느낄수있는 봄여행을 좋아함.

*주제2:여행갈 때 정보
- 계절마다 봄에는 벚꽃이 있는 화사한곳이나 동물원(전주), 가을에는 현충사와 같은 곳을 선호하며 여름과 겨울은 여행을 잘 다니지 않음.
- 홈페이지에서 시, 축제정보를 얻게된다면 도움이 될것같지만 글이 많으면 잘 안읽을것 같으니 진지하지 않고 사진이 많았으면 좋겠음.


*기본프로필
- A팀 시각디자인학과 11학번 김범선

*주제1:여행의 목적
- 여행지 선정할때 기준은 친구와 함께갈때 페이스북을 통해 3일전부터 계획하면서 맛집을 알아봄.
- 여행의 테마가 유적지라면 멀어서 그렇지 불국사와 다보탑과 같은 건축물을 보면서 옛것의 아름다움을 느낄 수 있어서 좋음.
- 선호하는 여행 테마는 무전여행의 경험이 있어 자유여행과 즉흥여행을 선호함.

*주제2:여행갈 때 정보
- 계절마다 봄과 가을에는 놀이공원, 여름에는 계곡, 겨울에는 따뜻함을 느낄 수 있는 바다를 선호함.

*기본프로필
- A팀 시각디자인학과 11학번 최민용

*주제1:여행의 목적
- 여행지 선정은 다른사람의 여행 후기글, 사진을 참고하며 자차가 없어 비용적인 부분을 많이 따지는 편.
- 여행의 테마가 유적지라면 과거사에 관심이 없어 흥미도 없으며. 유적지와 시를 떠올리니까 어둡고 딱딱한 느낌이 들어 어떻게 소화시킬지 잘 모르겠음.
- 선호하는 여행테마는 맛집탐방.

*주제2:여행갈 때 정보
- 봄과 가을에는 에버랜드, 여름에는 계곡이나 바다, 겨울에는 성안길(천안)과 같은 도시느낌이 나는 여행지를 좋아함.
- 시는 잘 몰라서 호불호가 갈릴듯하고, 음악은 여행지마다 클래식, 락 등 다르게 넣으면 좋을듯함.

*기본프로필
- A팀 시각디자인학과 11학번 김병기

*주제1:여행의 목적
- 여행지 선정은 기분에 따라 정하는편이며 혼자 배낭싸서 기차여행을 가는 즉흥여행을 좋아함.
- 유적지는 수학여행으로 많이 다녀봐서 별로이고. 사람많은 여행지를 안좋아함.
- 선호하는 여행테마는 혼자여행이며 여행을 통해 생각을 비우기도 함.

*주제2:여행갈 때 정보
- 딱딱한 시 말고 현대적이고 동요적인 시를 통해 스토리텔링을 하면 좋을듯함.
- 유명한 여행지가 아닌 곳을 홍보해줬으면 좋겠음.
